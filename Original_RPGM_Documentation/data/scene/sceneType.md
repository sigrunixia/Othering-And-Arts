# sceneType

The sceneType is a key that identifies the type of [scene](../../components/scene.md)

This value is valid if you use the [Scene Analyser](../../analyser/index.md).

Accepted Values:
- 'action'
- 'combat'
- 'encounter'
- 'exposition'
- 'investigation'
- 'planning'
- 'preparation'
- 'recap'
- 'socialcombat'
